# ServerController [![Build Status](https://travis-ci.org/SE7-KN8/ServerController.svg?branch=master)](https://travis-ci.org/SE7-KN8/ServerController) [![GitHub release](https://img.shields.io/github/release/se7-kn8/servercontroller.svg)](https://github.com/SE7-KN8/ServerController/releases)
A program to control servers (e.g. Spigot, Vanilla, BungeeCord).

## Getting Started
To install the ServerController on your computer read the [Installation guide](https://github.com/SE7-KN8/ServerController/wiki).

### Prerequisites
To run the ServerController you must have a valid Java 8 installation on your computer.

## Features
The current features of the ServerController are
* Controlling of Minecraft-Vanilla, CraftBukkit, Spigot, or BungeeCord servers
* RCon-Client
* Full addon-api to develop plugins for custom server-types
* Rest-Api to control servers remotely
* Editing server-files directly in the gui
* (Only for Minecraft) View server properties and operators
* A android app to control the servers from a smartphone
